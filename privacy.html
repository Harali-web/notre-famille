Politique de confidentialité — Notre famille

Dernière mise à jour : juillet 2026

Quelles données sont enregistrées ?

L'application « Notre famille » enregistre les informations que vous saisissez vous-même : noms, dates, photos, notes, récits, enregistrements audio/vidéo et statistiques d'utilisation (séries, niveaux, badges). Ces informations sont fournies volontairement par vous ou les membres de votre famille pour construire votre arbre généalogique.

Où sont stockées ces données ?

Par défaut, toutes les données restent uniquement sur votre appareil (stockage local du navigateur / de l'application), dans un espace propre à chaque compte famille (nom + mot de passe). Aucune donnée n'est envoyée à un serveur, sauf si vous activez explicitement une synchronisation en ligne (fonctionnalité optionnelle, non activée par défaut).

Le mot de passe du compte famille

Le mot de passe créé pour un compte famille protège uniquement l'accès sur l'appareil utilisé (comme un digicode local). Il n'est pas transmis sur internet et ne constitue pas un compte en ligne sécurisé au sens bancaire. Ne réutilisez pas un mot de passe sensible que vous utilisez ailleurs.

Partage de données

Vous pouvez choisir d'exporter, partager ou imprimer des données (livret, carte-image, message WhatsApp). Ces partages sont déclenchés uniquement par une action volontaire de votre part. L'application ne partage jamais vos données automatiquement.

Notifications

Si vous activez le rappel quotidien, l'application utilise les notifications de votre navigateur pour vous envoyer un rappel local. Aucune donnée n'est transmise à un tiers pour cela.

Suppression des données

Vous pouvez supprimer vos données à tout moment depuis l'application (bouton « Tout effacer ») ou en désinstallant l'application / en effaçant les données du site depuis les réglages de votre navigateur ou téléphone.
